# LOOM 项目交付文档 (Final Walkthrough - Premium Edition)

这是一个简约且**极其精美**的 Coze 智能体平台，已经完成了从后端引擎到**全链路视觉重塑**的闭环。

## 🌟 核心功能与视觉亮点

### 1. 极致视觉体验 (Premium UI/UX)
- **设计系统**：采用 `Inter` & `Outfit` 现代字体，基于 `Glassmorphism` (毛玻璃) 风格设计。
- **动效反馈**：全局实现了细腻的平滑过渡（Transitions）和交互反馈。
- **响应式布局**：全新设计的顶部导航栏和沉浸式工作区。

### 2. 可视化工作流编排 (Workflow Editor)
- **毛玻璃侧边栏**：点击节点展示精致的半透明配置面板。
- **实时日志看板**：底部深色控制台，实时追踪 Node 执行状态。
- **画布操作**：基于 `Vue Flow`，支持流式节点管理。

### 3. RAG 知识库中心 (Knowledge Hub)
- **卡片式管理**：采用大卡片布局，清晰展示文档数量与状态。
- **极简上传**：一键上传，后端自动完成分片与 Mock 索引。

### 4. 沉浸式 AI 对话 (Chat Experience)
- **气泡对话框**：右下角悬浮设计，支持消息气泡与打字机状态。
- **Agent 集成**：对话直接驱动底层工作流，响应灵活。

## 🛠️ 如何启动
1. 确保已在该目录下。
2. 双击 **`start.bat`**。
3. 等待两个黑色窗口（后端+前端）启动完毕。
4. 访问 `http://localhost:5173`。
